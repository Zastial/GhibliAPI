# GhibliAPI

Author: Alexandre CAROL

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run start:dev
```

API runs at `http://localhost:3000/api`  
Documentation at `http://localhost:3000/api/docs`

## Authentication

**Default admin key:** `admin-ghibli-api-key-dev-only`
